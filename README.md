## Setup & Installation

1. Clone the repository
   ```
   git clone https://github.com/AdnanMahida/ProductListDemo.git 
   ```

2. Set Current Working dir to productlistdemo

3. Run these commands
   ```
    flutter pub get
    flutter pub upgrade
    flutter pub run   
   ```
